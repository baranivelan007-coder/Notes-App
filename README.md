# Notes App

A simple user-specific notes application built to practice full-stack web development concepts.  
This project helped me understand how frontend and backend work together in a real application.

---

## Features
- User authentication
- Create, edit, and delete notes
- Notes are private to each user
- Simple and clean UI

---

## Tech Stack used

**Frontend**
- React.js
- Tailwindcss

**Backend**
- Express.js

**Database**
- MongoDB

---

## Purpose of This Project
This project was built to:
- Apply concepts I learned in full-stack development
- Understand CRUD operations
- Learn how authentication and backend APIs work
- Practice connecting frontend with backend
